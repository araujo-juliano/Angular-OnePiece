Nami

Nami é a navegadora dos Chapéus de Palha em One Piece, uma talentosa cartógrafa e ladra habilidosa. Seu sonho é criar um mapa completo do mundo. Inicialmente, Nami se juntou à tripulação como uma cativa, buscando reunir dinheiro para libertar sua vila de uma ameaça pirata. Após a resolução desse conflito, ela decidiu permanecer com os Chapéus de Palha, onde se tornou uma figura essencial na navegação e planejamento estratégico da tripulação, continuando a buscar sua paixão pela cartografia.

Log Pose

O Log Pose é uma bússola especial que se adapta às condições magnéticas únicas de cada ilha na Grand Line. Ao invés de apontar para o norte magnético como uma bússola comum, o Log Pose é ajustado para apontar para a próxima ilha na rota pré-determinada.
